我的第一篇博客-搭建博客

1.安装node.js

下载：网址nodejs.org LTS版本 长期支持版

2.cmd下安装环境

node -v
npm -v
npm install -g cnpm –registry=https://registry.npm.taobao.org
cnpm -v
cnpm install -g hexo-cli
hexo -v
cd c:
mkdir blog
cd blog
chdir
hexo init
hexo g
dir
hexo s

3.写博客

hexo n “我的第一篇博客-搭建博客”
可以在notepad++ 里面编辑博客

4.登录github部署blog

https://github.com/wangyiwang01
右上角 new repository

  • repository name : wangyiwang01.github.io (github仓库只能是这个地址)

blog文件夹下
cnpm install –save hexo-deployer-git

打开_config.yml修改如下配置

Deployment

Docs: https://hexo.io/docs/one-command-deployment

deploy:
type: git
repo: https://github.com/wangyiwang01/wangyiwang01.github.io.git
branch: master

hexo d
